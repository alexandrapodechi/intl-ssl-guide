# intl-ssl-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面主要围绕 SSL/TLS 相关的技术说明、配置参考与操作指南，不针对任何特定域名或具体网站。

## 项目简介

intl-ssl-guide 是一个静态 HTML 页面集合仓库，旨在以文件形式保存与 SSL/TLS 协议、证书管理、安全配置等主题相关的独立文档。每个 HTML 页面均可单独查看或部署，适合作为技术参考或离线查阅资料。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 索引页面（可选）
└── pages/             # 存放独立 HTML 页面
    ├── page-1.html
    ├── page-2.html
    └── ...
```

- `pages/` 目录下存放所有归档的 HTML 文件，文件名建议反映页面内容主题。
- 根目录下的 `index.html` 可作为入口页，提供页面列表或导航。

## 页面归档说明

- 每个 HTML 文件均为独立完整的文档，不依赖于其他页面或外部资源（除通用 CSS/JS 外）。
- 页面内容以技术说明、配置示例、操作步骤为主，不包含营销性文字或诱导性内容。
- 归档页面可能来自不同时期，内容仅供参考，使用前请自行验证其适用性。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面时保持目录结构清晰。
- 欢迎通过 Issue 或 Pull Request 提出改进建议或提交新的归档页面。
- 提交内容请确保不包含具体域名、个人隐私信息或商业推广元素。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开任意 HTML 文件查看。
3. 也可将 `pages/` 目录部署到任意静态站点服务中。
